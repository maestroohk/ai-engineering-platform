# `prompts/` — moved

> **This directory is intentionally empty.**
>
> The prompt templates that previously lived here have been
> moved to [`.ai/prompts/`](./.ai/prompts/). The canonical home
> for AI collaboration documents is [`.ai/`](./.ai/).
>
> This directory is preserved as a redirect so that external
> bookmarks, search-engine results, and old commit messages
> continue to resolve. It is **not** a working location.
>
> If you are looking for a prompt template, go to
> [`.ai/prompts/`](./.ai/prompts/).
>
> If you are looking for the entry point to AI-assisted
> development, read [`.ai/README.md`](./.ai/README.md) first,
> then [`.ai/session-start.md`](./.ai/session-start.md).
>
> The precedence hierarchy (from
> [`AGENTS.md`](./AGENTS.md) § 2.2) is:
>
> 1. `AGENTS.md`
> 2. `DECISIONS.md`
> 3. `ARCHITECTURE.md` and `STYLEGUIDE.md`
> 4. `docs/`
> 5. `.ai/workflows/`
> 6. `.ai/prompts/`
> 7. Individual task instructions
>
> This directory will be removed entirely once the migration
> is complete across all external links.
