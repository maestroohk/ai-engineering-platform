# `wwwroot/css/`

This directory holds the **compiled** output of the Tailwind
pipeline. `app.css` is regenerated by `npm run css:build` (or
`npm run css:watch` while developing); it is not hand-edited
and is excluded from version control.

The pipeline's source-of-truth files live at
`src/AiEng.Platform.App/Styles/`:

- `input.css` — the Tailwind entry file
- `tokens.css` — non-colour CSS custom properties
- `themes.css` — colour variables for light / dark themes and
  the OS-preference fallback
- `base.css` — body, type, focus, scrollbar, reduced-motion
- `utilities.css` — a small set of project-specific `@apply`
  utilities

Edit the source files; never edit `app.css` directly.
